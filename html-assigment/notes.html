SOME THINGS THAT I LEARN IN 5 WEEKS ABOUT HTML

The most important, the differences between the Tags and Attributes.

  • <Tags>: (The Structure) These are the building blocks that tell the browser what kind of element to create. Example: <h1>
  • <Attributes>: (The Details) These live inside the opening tag and provide specific settings. Example: src="image.jpg"
  • <a>: (The Anchor) Used to create clickable hyperlinks. Example: <a href="url">Link</a>
  • <img>: (The Image) A self-closing tag used to embed pictures. Example: <img src="file.png">
  • <div>: (The Division) A container used to group elements together. Example: <div style="color:red">
